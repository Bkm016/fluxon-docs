# Fluxon Mintlify Docs

This folder contains Fluxon's documentation site powered by Mintlify.

## Local preview

From `mintlify-docs/`:

```bash
pnpm install
pnpm exec mint dev
```

Then open `http://localhost:3000`.

## Contributing

- Site navigation lives in `mintlify-docs/docs.json`.
- Writing standards and sync checklist: `mintlify-docs/developer/documentation.mdx`.
